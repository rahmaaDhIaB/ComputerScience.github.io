Q1 Linear Search — count occurrences & find first

(a)(i) Use a FOR loop through all elements. Initialise count = 0 before the loop. Add 1 each time you find a match. Return count after the loop.

(b)(i) Use a WHILE loop with two conditions: i < size AND not found. Set found = True on a match. Increment i only in the ELSE branch so you don't skip the index.

(a)(i) — count_code()  [3 marks]
# Global variables (declared at the top of the program)
codes = [1042, 2081, 3017, 2081, 4055, 2081, 1042, 3300]
size  = 8


def count_code(target):
    # Initialise counter to zero before starting the search
    count = 0

    # Loop through every element in the array using its index
    for i in range(size):

        # Check if the current element matches what we are looking for
        if codes[i] == target:
            count += 1    # increment counter each time a match is found

    # Return the total count after checking all elements
    return count

(b)(i) — find_first()  [5 marks]
def find_first(target):
    # Start at index 0 — the beginning of the array
    i = 0

    # Flag to track whether we have found the target
    found = False

    # Continue searching while there are elements left AND we have not found it yet
    # Using 'not found' allows the loop to exit early as soon as a match is found
    while i < size and not found:

        if codes[i] == target:
            # Match found — set flag to True so the while loop exits
            found = True

        else:
            # No match at this index — move to the next one
            # Important: we only increment i in the ELSE branch
            # so that 'i' still holds the correct index when we exit
            i += 1

    # After the loop: check if we actually found it or ran out of elements
    if found:
        return i       # return the index where target was first found
    else:
        return -1     # -1 signals "not found" — no valid index is ever -1

Binary search: set low = 0 and high = size - 1. In each iteration calculate mid = (low + high) // 2. If marks[mid] equals the target — found. If target is greater, move low up. If target is smaller, move high down. The loop condition must be low <= high — using just < misses the last element.

(b)(i) find_all() must use a FOR loop — never a while with early exit — because it needs to check every single element to find all matches.
(a)(i) — binary_search()  [6 marks]
# Global variables
marks = [5, 12, 18, 24, 31, 47, 58, 63, 79, 91]
size  = 10


def binary_search(target):
    # Set the initial boundaries of the search region
    low  = 0            # start of the region we are searching
    high = size - 1    # end of the region we are searching

    # Keep searching while the region is valid (low must not pass high)
    # IMPORTANT: use <= not < so we don't skip the last remaining element
    while low <= high:

        # Calculate the middle index of the current region
        mid = (low + high) // 2

        if marks[mid] == target:
            # Exact match — return the index immediately
            return mid

        elif marks[mid] < target:
            # Target is in the RIGHT half — discard the left half
            low = mid + 1

        else:
            # Target is in the LEFT half — discard the right half
            high = mid - 1

    # If we exit the loop without returning, the target was not found
    return -1

def find_all(target):
    # Start with an empty list — we will add indices to it as we find matches
    found_list = []

    # Use a FOR loop — we MUST check every element to find ALL occurrences
    # We cannot use a while with early exit here, unlike find_first()
    for i in range(size):

        if marks[i] == target:
            # Match found — record this index in the list
            found_list.append(i)

    # Return the list of all matching indices (may be empty if no match)
    return found_list

swapped flag pattern:
— Set swapped = True once BEFORE the outer loop so you enter it at least once.
— At the START of every pass, reset swapped = False.
— Inside the swap block, set swapped = True.
— Use if not swapped: break after the inner loop to exit early.

Swap — always use a temporary variable:
temp = scores[j] → scores[j] = scores[j+1] → scores[j+1] = temp

Inner loop upper bound: After each pass the largest element is in its final place, so the inner loop can be range(size - 1 - i).
(a)(i) — print_scores()  [2 marks]
# Global variables
scores = [64, 12, 45, 8, 99, 33]
size   = 6


def print_scores():
    # Loop through all elements and print each on the same line
    for i in range(size):
        # end=" " prevents the default newline so all values stay on one line
        print(scores[i], end=" ")

    # Print a newline at the end so the next output starts on a new line
    print()
(a)(ii) — bubble_sort() with swapped flag  [6 marks]
def bubble_sort():
    # Outer loop controls the number of passes through the array
    # After each pass, the largest unsorted element is in its correct place
    for i in range(size - 1):

        # Assume no swaps will be made during this pass
        swapped = False

        # Inner loop compares adjacent pairs
        # Upper bound reduces by i each pass — last i elements are already sorted
        for j in range(size - 1 - i):

            if scores[j] > scores[j + 1]:
                # These two elements are in the wrong order — swap them

                # Step 1: save the left element before overwriting it
                temp = scores[j]

                # Step 2: move the right element to the left position
                scores[j] = scores[j + 1]

                # Step 3: place the saved value into the right position
                scores[j + 1] = temp

                # A swap was made — we cannot stop yet
                swapped = True

        # If no swap happened during the entire pass, the array is already sorted
        # We can exit the outer loop early — no more passes needed
        if not swapped:
            break

# To sort in DESCENDING order, change ONE line in bubble_sort():
#
# Change:  if scores[j] > scores[j + 1]:
# To:      if scores[j] < scores[j + 1]:
#
# Reason: we now swap when the LEFT element is SMALLER than the right,
# which moves larger values to the left instead of the right.

Insertion sort — the three key steps per iteration:
1. Save key = temps[i] — this is the element we are going to insert.
2. Set j = i - 1 — start scanning left from the position before i.
3. WHILE j >= 0 AND temps[j] > key: shift temps[j] one place right, then move j left.
4. After the while loop: place key at temps[j + 1].

find_min_max(): After insertion_sort() has been called, the list is already sorted. So temps[0] is the minimum and temps[size - 1] is the maximum — no loop needed.
(a)(i) — insertion_sort()  [6 marks]
# Global variables
temps = [29, 4, 15, 8, 22, 37, 11]
size  = 7


def insertion_sort():
    # Start from index 1 — the element at index 0 is already "sorted" on its own
    for i in range(1, size):

        # Save the current element — this is the one we need to insert correctly
        key = temps[i]

        # Start comparing with the element just to the left of i
        j = i - 1

        # Shift elements that are larger than key one position to the right
        # This creates the gap where we will insert key
        # Condition 1: j >= 0 — don't go past the beginning of the array
        # Condition 2: temps[j] > key — only shift if the element is larger
        while j >= 0 and temps[j] > key:

            # Move this element one place to the right
            temps[j + 1] = temps[j]

            # Move one step further to the left
            j -= 1

        # The while loop stopped — j+1 is now the correct position for key
        # (either we reached the beginning, or we found a smaller element)
        temps[j + 1] = key

def find_min_max():
    # After insertion_sort() has been called, temps is sorted in ascending order
    # So the minimum is always at index 0 and the maximum at the last index
    # No loop is needed — we can read them directly

    print(f"Smallest value: {temps[0]}")
    print(f"Largest value:  {temps[size - 1]}")
