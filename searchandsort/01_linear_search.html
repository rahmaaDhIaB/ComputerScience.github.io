What is Linear Search?
The simplest searching algorithm — no tricks, just check each one
🧠 The analogy You lost your pen somewhere on a long shelf of books. You start at the left end and check each book one by one — "is the pen here? no. here? no. here? YES!" — you found it at position 3. That is literally linear search. It works on any list, sorted or not, because it checks everything.
⚠️ The catch If the list has 1000 items and your value is at the very end (or not there at all), you must check all 1000. For large lists this is slow. That is why Binary Search exists — but that requires a sorted list.

The Python Code 🐍
Read every comment — each line maps to a step in the visual above
def linear_search(my_list, search_value):

    max_index = len(my_list) - 1   # index of the very last element
    found     = False                # flag — not found yet
    index     = -1                   # start BEFORE position 0

    while not found and index < max_index:
        index += 1                   # move pointer one step right
        if my_list[index] == search_value:
            found = True            # match! loop will stop

    if found:
        print("Value found at location:", index)
    else:
        print("Value not found")
🔎 Every variable explained
max_index
→
The last valid index (length − 1). The loop must not go past this — doing so would cause an IndexError.
found
→
Starts as False. Becomes True the moment we find a match. This stops the while loop immediately — we don't keep searching after we've already found it.
index = −1
→
We start at −1 so that the very first thing the loop does (index += 1) moves us to position 0 — the real start of the array.
while ... and ...
→
TWO conditions must BOTH be true to keep looping: (1) we haven't found it yet, AND (2) we haven't gone past the last element. Either one being false exits the loop.

The loop condition checks TWO things: not found AND index < max_index. Since 6 is not in the list, found never becomes TRUE. The loop exits when index reaches 4 (= max_index), because the condition index < 4 becomes FALSE.

E3 Write linear_search() from scratch
Intermediate [5 marks] ▼ expand
A program stores up to 20 student names in the global 1D array Names. The global variable NumNames stores how many names are in the array. Write the function LinearSearch() that takes a string parameter SearchName, performs a linear search on Names, and returns the index if found or −1 if not found.
  • Write program code for LinearSearch(). [5]
Cambridge Paper 4 style. The function must RETURN the index (not just print it).
Structure: declare found=False, index=-1, max_index=NumNames-1. Loop while not found and index < max_index: increment index, check Names[index]. After loop: if found return index, else return -1. 
def LinearSearch(SearchName):
    global Names, NumNames

    max_index = NumNames - 1
    found     = False
    index     = -1

    while not found and index < max_index:
        index += 1
        if Names[index] == SearchName:
            found = True

    if found:
        return index
    else:
        return -1

# Mark scheme (5 marks):
# [1] max_index = NumNames - 1 (or equivalent)
# [1] found and index correctly initialised
# [1] while loop with correct double condition
# [1] index incremented before checking Names[index]
# [1] return index if found, return -1 if not
