The 4 Rules — Tattoo These On Your Brain
memorise or fail
-1 means STOP 🛑
  • pointer = -1 → no next item (end of list)
  • FirstNode = -1 → list is EMPTY
  • FirstEmpty = -1 → list is FULL (no space)
Always Check Full First ✅
  • Before ANY insert: check FirstEmpty == -1
  • If full → print message → return
  • Forget this = lose marks instantly
Two-Pointer Delete 🔗
  • Need current AND previous
  • previous starts at -1
  • Skip over: prev → next of current
Recycle After Delete ♻️
  • Clear the slot's data
  • Point deleted slot to old FirstEmpty
  • Set FirstEmpty = deleted slot
📦
What's a Node?
one slot = two things
Imagine a Post-it with two sections:
Left side = the actual value (10, "Alice", whatever)
Right side = slot number of the NEXT item
That's it. Two columns. Done.

Insert at Front
most common exam question
1
Check full: if FirstEmpty == -1 → print message, return
2
Grab slot: new_slot = FirstEmpty
3
Move free pointer: FirstEmpty = LinkedList[FirstEmpty][1]
4
Store data: LinkedList[new_slot][0] = value
5
Point to old head: LinkedList[new_slot][1] = FirstNode
6
Update head: FirstNode = new_slot
def InsertFront(value):
    global LinkedList, FirstNode, FirstEmpty
    if FirstEmpty == -1:
        print("List is full"); return
    new_slot                = FirstEmpty
    FirstEmpty              = LinkedList[FirstEmpty][1]
    LinkedList[new_slot][0] = value
    LinkedList[new_slot][1] = FirstNode
    FirstNode               = new_slot

Traverse / Output
walk the chain
It's literally just following arrows
Start at FirstNode. Follow [1] until you hit -1. Print [0] each time. That's the entire algorithm.
1
current = FirstNode
2
while current != -1: keep going
3
print LinkedList[current][0]
4
current = LinkedList[current][1] → follow arrow
def OutputList():
    global LinkedList, FirstNode
    current = FirstNode
    while current != -1:
        print(LinkedList[current][0])
        current = LinkedList[current][1]
Count items? Same loop + count += 1 each iteration, then return count.
🔍
Search / Find
traverse + check
Same as traverse but with an if inside
Walk the list. At each step: is this the one I'm looking for? Yes → done. No → keep walking. 
def SearchList(target):
    global LinkedList, FirstNode
    current = FirstNode
    while current != -1:
        if LinkedList[current][0] == target:
            print("Found: slot", current); return
        current = LinkedList[current][1]
    print("Not found")
Don't forget: "Not found" goes AFTER the loop, not inside it!

Delete — The Boss Level
hardest but most marks
The Skip-Over Trick Visualised
Before: A → B → C   (deleting B)
After:   A → C          (A skips B)
Code:   prev[1] = current[1]
1
Two walkers: current = FirstNode, previous = -1
2
Walk until you find target (move BOTH forward each step)
3
If previous == -1: deleting head → FirstNode = current[1]
4
Else: LinkedList[previous][1] = LinkedList[current][1]
5
Recycle: clear data, point slot to old FirstEmpty, FirstEmpty = current
def DeleteItem(target):
    global LinkedList, FirstNode, FirstEmpty

    current  = FirstNode
    previous = -1

    # Walk until we find it
    while current != -1 and \
          LinkedList[current][0] != target:
        previous = current
        current  = LinkedList[current][1]

    # Not found?
    if current == -1:
        print("Not found"); return

    # Re-link (skip over deleted node)
    if previous == -1:
        FirstNode = LinkedList[current][1]
    else:
        LinkedList[previous][1] = \
            LinkedList[current][1]

    # Recycle the slot back to free list
    LinkedList[current][0] = -1
    LinkedList[current][1] = FirstEmpty
    FirstEmpty = current
Don't forget to recycle! 3 lines at the end. Easy marks that students always miss.

Mark Scheme Patterns
examiner brain decoded
OperationWhat marks test
InsertFull check · grab slot · advance FirstEmpty · store data · point to old head · update head
TraverseStarts at FirstNode · loop until -1 · prints [0] · advances via [1]
SearchSame as traverse · checks [0]==target · returns/prints slot · "Not found" after loop
DeleteTwo pointers · loop condition · moves both · handles head case · skips over · recycles
Pattern: Every function starts the same: global ... then current = FirstNode. When in doubt, write that first.

Don't Be That Student
common fails

❌
Only one pointer in delete — need both current AND previous
❌
Forgot to recycle the slot after delete (3 lines, easy marks)
❌
Put "Not found" inside the loop → prints wrong number of times
❌
Forgot to declare global → function won't see the variables
❌
Using [0] and [1] backwards — [0] = DATA, [1] = NEXT POINTER
