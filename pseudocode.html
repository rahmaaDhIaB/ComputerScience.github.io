Translate the pseudocode exactly into Python. The program converts Celsius to Fahrenheit.
Cambridge Pseudocode
DECLARE celsius : REAL
DECLARE fahrenheit : REAL

OUTPUT "Enter temperature in Celsius:"
celsius ← USERINPUT

fahrenheit ← (celsius * 9 / 5) + 32

OUTPUT "Fahrenheit: " & fahrenheit
Your Python
# Write your Python here

P2 Ticket Pricing
Easy [5 marks] ▼ expand
A cinema charges different prices depending on the customer's age. Translate this decision logic to Python.
Cambridge Pseudocode
DECLARE age : INTEGER
DECLARE price : REAL

OUTPUT "Enter age:"
age ← USERINPUT

IF age < 12 THEN
    price ← 5.00
ELSE IF age >= 65 THEN
    price ← 6.50
ELSE
    price ← 10.00
ENDIF

OUTPUT "Ticket price: $" & price
Your Python
# Write your Python here
Cambridge note: ELSE IF in pseudocode becomes elif in Python. The ENDIF is replaced by dedenting.
age = int(input("Enter age: ")) if age < 12: price = 5.00 elif age >= 65: price = 6.50 else: price = 10.00 print(f"Ticket price: ${price}")
P3 Countdown Loop
Easy [5 marks] ▼ expand
A program uses both a FOR loop and a WHILE loop. Translate both correctly — pay attention to how the loops end.
Cambridge Pseudocode
// Part 1: FOR loop
FOR i ← 10 TO 1 STEP -1
    OUTPUT i
NEXT i
OUTPUT "Liftoff!"

// Part 2: WHILE loop
DECLARE total : INTEGER
total ← 0
DECLARE n : INTEGER
n ← 1
WHILE n <= 5 DO
    total ← total + n
    n ← n + 1
ENDWHILE
OUTPUT "Sum 1 to 5: " & total
Your Python
# Write your Python here
Tricky bit: FOR i ← 10 TO 1 STEP -1 in pseudocode counts down. In Python: range(10, 0, -1) — the step is the third argument.
# Part 1 for i in range(10, 0, -1): print(i) print("Liftoff!") # Part 2 total = 0 n = 1 while n <= 5: total += n n += 1 print(f"Sum 1 to 5: {total}")

P4 Number Classifier Procedure
Intermediate [6 marks] ▼ expand
A procedure (no return value) classifies a number as positive, negative or zero. Translate the full program including the procedure call.
Cambridge Pseudocode
PROCEDURE Classify(num : INTEGER)
    IF num > 0 THEN
        OUTPUT num & " is positive"
    ELSE IF num < 0 THEN
        OUTPUT num & " is negative"
    ELSE
        OUTPUT "zero"
    ENDIF
ENDPROCEDURE

// Main program
DECLARE x : INTEGER
FOR i ← 1 TO 3
    OUTPUT "Enter a number:"
    x ← USERINPUT
    CALL Classify(x)
NEXT i
Your Python
# Write your Python here
Cambridge note: CALL Classify(x) in pseudocode simply becomes Classify(x) in Python — no CALL keyword needed. A procedure has no return statement.
def Classify(num): if num > 0: print(f"{num} is positive") elif num < 0: print(f"{num} is negative") else: print("zero") for i in range(3): x = int(input("Enter a number: ")) Classify(x)
P5 Array Search Function
Intermediate [8 marks] ▼ expand
A function performs a linear search on a 1D array and returns the index of the target, or -1 if not found.
Cambridge Pseudocode
FUNCTION LinearSearch(
    arr : ARRAY, 
    target : INTEGER) 
    : INTEGER

    DECLARE i : INTEGER
    FOR i ← 0 TO LEN(arr) - 1
        IF arr[i] = target THEN
            RETURN i
        ENDIF
    NEXT i
    RETURN -1
ENDFUNCTION

// Main program
DECLARE nums : ARRAY[0:5] OF INTEGER
nums ← [14, 7, 33, 9, 21, 5]

OUTPUT LinearSearch(nums, 9)
OUTPUT LinearSearch(nums, 99)
Your Python
# Write your Python here
Cambridge note: arr[i] = target in pseudocode is comparison, not assignment. In Python, comparison is == not =. This is a very common exam mistake!
def LinearSearch(arr, target): for i in range(len(arr)): if arr[i] == target: # == not = return i return -1 nums = [14, 7, 33, 9, 21, 5] print(LinearSearch(nums, 9)) # → 3 print(LinearSearch(nums, 99)) # → -1
P6 String Processing — Username Generator
Intermediate [8 marks] ▼ expand
A school generates usernames from a student's first name, last name, and year of birth. Translate the string manipulation logic carefully.
Cambridge Pseudocode
FUNCTION MakeUsername(
    first   : STRING,
    last    : STRING,
    year    : INTEGER) : STRING

    DECLARE username : STRING
    username ← LCASE(
        SUBSTRING(first, 0, 1) &
        SUBSTRING(last,  0, 3) &
        STRING(year MOD 100)
    )
    RETURN username
ENDFUNCTION

// Main program
DECLARE f, l : STRING
DECLARE y : INTEGER
OUTPUT "First name:"
f ← USERINPUT
OUTPUT "Last name:"
l ← USERINPUT
OUTPUT "Year of birth:"
y ← USERINPUT
OUTPUT MakeUsername(f, l, y)
Your Python
# Write your Python here
Key translations:
• SUBSTRING(s, 0, 1) → s[0:1] (first 1 char)
• SUBSTRING(s, 0, 3) → s[0:3] (first 3 chars)
• LCASE(s) → s.lower()
• STRING(n) → str(n)
• String concatenation & → + in Python
def MakeUsername(first, last, year): username = (first[0:1] + last[0:3] + str(year % 100)).lower() return username f = input("First name: ") l = input("Last name: ") y = int(input("Year of birth: ")) print(MakeUsername(f, l, y)) # e.g. "Ali", "Hassan", 2007 → "ahas07"

P7 File Reading with Exception Handling
Hard [10 marks] ▼ expand
Directly based on Question 2 of the exam you uploaded. A procedure reads lines from a file, counts them, and handles errors. Translate precisely.
Cambridge Pseudocode
PROCEDURE ReadFile(filename : STRING)
    DECLARE line : STRING
    DECLARE count : INTEGER
    count ← 0

    TRY
        OPEN filename FOR READ
        WHILE NOT EOF(filename) DO
            line ← READLINE(filename)
            count ← count + 1
            OUTPUT count & ": " & line
        ENDWHILE
        CLOSE filename
    EXCEPT
        OUTPUT "Error: could not read file"
    ENDTRY

    OUTPUT "Lines read: " & count
ENDPROCEDURE

// Main program
DECLARE fname : STRING
OUTPUT "Enter filename:"
fname ← USERINPUT
CALL ReadFile(fname)
Your Python
# Write your Python here
Key translations:
• OPEN / CLOSE / READLINE → use with open(..., "r") as f: (auto-closes)
• WHILE NOT EOF → for line in f: (Python handles EOF automatically)
• TRY / EXCEPT / ENDTRY → try: / except:
• Use line.strip() to remove newline characters
def ReadFile(filename): count = 0 try: with open(filename, "r") as f: for line in f: count += 1 print(f"{count}: {line.strip()}") except: print("Error: could not read file") print(f"Lines read: {count}") fname = input("Enter filename: ") ReadFile(fname)
P8 Recursive Power Function + OOP Class
Hard [12 marks] ▼ expand
Two parts: a recursive function AND a class declaration. Both appear regularly in Paper 4. Translate each part independently then combine them in the main program.
Cambridge Pseudocode
// Part 1: recursive function
FUNCTION Power(base : INTEGER,
               exp  : INTEGER) : INTEGER
    IF exp = 0 THEN
        RETURN 1
    ELSE
        RETURN base * Power(base, exp - 1)
    ENDIF
ENDFUNCTION

// Part 2: class
CLASS Calculator
    PRIVATE memory : REAL

    PUBLIC PROCEDURE NEW()
        memory ← 0.0
    ENDPROCEDURE

    PUBLIC PROCEDURE Store(val : REAL)
        memory ← val
    ENDPROCEDURE

    PUBLIC FUNCTION Recall() : REAL
        RETURN memory
    ENDFUNCTION
ENDCLASS

// Main program
calc ← NEW Calculator()
CALL calc.Store(Power(2, 8))
OUTPUT calc.Recall()
Your Python
# Write your Python here
Key translations:
• CLASS ... ENDCLASS → class Name: with indentation
• PUBLIC PROCEDURE NEW() → def __init__(self):
• PRIVATE memory → self.memory (Python uses self. — no true private, but use self.__memory for convention)
• calc ← NEW Calculator() → calc = Calculator()
• Recursive base case: if exp == 0: return 1
# Part 1 — recursive function def Power(base, exp): if exp == 0: return 1 else: return base * Power(base, exp - 1) # Part 2 — class class Calculator: # memory : Real def __init__(self): self.memory = 0.0 def Store(self, val): self.memory = val def Recall(self): return self.memory # Main program calc = Calculator() calc.Store(Power(2, 8)) print(calc.Recall()) # → 256.0
