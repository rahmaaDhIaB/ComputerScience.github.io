What is a Binary Search Tree?
Every node has a LEFT child (smaller values) and a RIGHT child (larger values)
🧠 The analogy — the 20-questions game Think of "guess the number between 1–100." You say 50. Too high? Go left (try 25). Too low? Go right (try 75). Each comparison eliminates HALF the remaining options. That's binary search tree logic — smaller values live on the left branch, larger values on the right branch.
🔑 The BST Property (memorise this!) For every node N: ALL values in the LEFT subtree < N. ALL values in the RIGHT subtree > N. This property is maintained on every insert — it's what makes searching fast.
🎯 Cambridge stores binary trees in a 2D array with 3 columns Column 0 = Data. Column 1 = Left child pointer (–1 if none). Column 2 = Right child pointer (–1 if none). A global variable RootNode stores the index of the root (–1 if empty tree).

02
Inserting a New Value
Always start at the root, go left if smaller, right if larger, until you find an empty spot
Example: Inserting 45 into the tree above (50 → 30 → 40 → right of 40)
1
Start at RootNode. Compare 45 with root value 50. 45 < 50 → go LEFT.
2
Now at node 30. Compare 45 with 30. 45 > 30 → go RIGHT.
3
Now at node 40. Compare 45 with 40. 45 > 40 → go RIGHT. Right pointer of 40 is –1 (empty!).
4
Found an empty spot! Store 45 at the next available index. Set node 40's RIGHT pointer to the new index.
class Node:
    def __init__(self, pNodeData):
        self._NodeData = pNodeData
        self._LeftNode = None
        self._RightNode = None

    def GetData(self):
        return self._NodeData

    def GetLeft(self):
        return self._LeftNode

    def GetRight(self):
        return self._RightNode

    def SetLeft(self, NewNode):
        self._LeftNode = NewNode

    def SetRight(self, NewNode):
        self._RightNode = NewNode


class Tree:
    def __init__(self, FirstNode):
        self._FirstNode = FirstNode

    def GetRootNode(self):
        return self._FirstNode

    # Insert from Cambridge past paper 9618_s25_qp_41
    def Insert(self, NewNode):
        CurrentNode = self._FirstNode
        Inserted = True
        while Inserted:
            if NewNode.GetData() < CurrentNode.GetData():
                if CurrentNode.GetLeft() == None:
                    CurrentNode.SetLeft(NewNode)
                    return True
                else:
                    CurrentNode = CurrentNode.GetLeft()
            else:
                if CurrentNode.GetRight() == None:
                    CurrentNode.SetRight(NewNode)
                    return True
                else:
                    CurrentNode = CurrentNode.GetRight()


def OutputInOrder(RootNode):
    if RootNode.GetLeft() != None:
        OutputInOrder(RootNode.GetLeft())
    print(RootNode.GetData())
    if RootNode.GetRight() != None:
        OutputInOrder(RootNode.GetRight())

04
Searching the Tree
Same logic as insert — follow the BST property at each node
Example: Searching for 40 in the tree. Path: 50 → 30 → 40 ✅
currentDataTargetComparisonMove
0504040 < 50Go LEFT → index 1
1304040 > 30Go RIGHT → index 4
4404040 == 40 ✅FOUND at index 4
def Search(target):
    global RootNode
    
    current = RootNode
    
    while current != None:
        if current.GetData() == target:
            print("Found:", target)
            return current
        elif target < current.GetData():
            current = current.GetLeft()
        else:
            current = current.GetRight()
    
    print("Not found")
    return None

3(c)(iii) Write Insert() — place a node in the correct position
Intermediate [6 marks] ▼ expand
The method Insert() belongs to the Tree class. It takes a Node object as a parameter and searches the tree to find the correct insertion position by: checking whether the new node's data is less than or greater than the current node's data; moving left if less, right if greater or equal; repeating until the correct empty position is found, then storing the node there.
  • Write program code for Insert(). [6]
Cambridge exam tip. Use a while loop (not recursion) for the walk. The mark scheme awards one mark each for: method header; checking left when data is less and attaching if null; checking right when data ≥ and attaching if null; the loop that continues until insertion.
Start at FirstNode. Loop forever: if NewNode.GetData() < CurrentNode.GetData() → check left; if left is None attach and return, else move left. Otherwise check right; if right is None attach and return, else move right.
def Insert(self, NewNode):
    CurrentNode = self._FirstNode
    Inserted = True
    while Inserted:
        if NewNode.GetData() < CurrentNode.GetData():
            if CurrentNode.GetLeft() == None:
                CurrentNode.SetLeft(NewNode)
                return
            else:
                CurrentNode = CurrentNode.GetLeft()
        else:
            if CurrentNode.GetRight() == None:
                CurrentNode.SetRight(NewNode)
                return
            else:
                CurrentNode = CurrentNode.GetRight()

# Mark scheme (6 marks):
# [1] Insert() method header taking one Node parameter
# [1] If new node data < current node → check left child
# [1] … attach to left if null / None
# [1] If new node data >= current node → check right child
# [1] … attach to right if null / None
# [1] Loop repeats until correct position is found (while loop / recursion)
3(d) Write OutputInOrder() — recursive in-order traversal
Intermediate [5 marks] ▼ expand
The recursive procedure OutputInOrder() outputs the binary tree data in ascending numerical order. It takes a Node object as a parameter. It: checks if the left node is null — if not, calls itself with the left node; outputs the current node's data; checks if the right node is null — if not, calls itself with the right node.
  • Write program code for OutputInOrder(). [5]
Note: unlike the array-based version, this OOP version does NOT check for –1. It checks if the left/right child is None before recursing — it does NOT recurse into None. The order is: recurse left if exists → print data → recurse right if exists.
def OutputInOrder(RootNode):
    if RootNode.GetLeft() != None:
        OutputInOrder(RootNode.GetLeft())
    print(RootNode.GetData())
    if RootNode.GetRight() != None:
        OutputInOrder(RootNode.GetRight())

# Mark scheme (5 marks):
# [1] Procedure header taking a Node as parameter (recursive)
# [1] Checking if left is null, and recursive call with left node if not null
# [1] Outputting current node's data
# [1] Checking if right is null, and recursive call with right node if not null
# [1] Correct order: left → output → right

class Tree:
    def __init__(self, pFirstNode):
        self._FirstNode = pFirstNode
    def GetRootNode(self): return self._FirstNode
    def Insert(self, NewNode):
        CurrentNode = self._FirstNode
        while True:
            if NewNode.GetData() < CurrentNode.GetData():
                if CurrentNode.GetLeft() == None:
                    CurrentNode.SetLeft(NewNode); return
                else: CurrentNode = CurrentNode.GetLeft()

                if CurrentNode.GetRight() == None:
                    CurrentNode.SetRight(NewNode); return
                else: CurrentNode = CurrentNode.GetRight()
